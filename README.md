# AI News Digest

Personalized AI news digest for your Telegram channel. Learns your writing style.

**What it does:** Fetches RSS feeds, selects top news with AI, writes a digest post **in your voice** (not generic AI text), and publishes to your Telegram channel.

## How it works

```
RSS/Atom feeds (configurable)
    |
    v
Filter (dedup + recency + keywords)
    |
    v
Haiku (selects 5-7 top news, structured JSON)
    |
    v
Sonnet/Opus (writes post in YOUR style + humanizer anti-AI rules)
    |
    v
Telegram channel
```

Key features:
- **Style learning** — paste your sample posts during setup, AI analyzes your voice and writes like you
- **Humanizer** — built-in anti-AI writing rules ([blader/humanizer](https://github.com/blader/humanizer), 25 patterns from Wikipedia's "Signs of AI writing")
- **Two-model pipeline** — fast model selects news, quality model writes. Saves cost.
- **Priority markers** — fire emoji for must-read items, bullet for regular
- **Topic packs** — preset feeds for AI/ML, startups, crypto, fintech, web dev, cybersecurity, product management

## Quickstart

### Option A: Claude Code (recommended)

If you have [Claude Code](https://docs.anthropic.com/en/docs/claude-code) with a Max or Team subscription:

```bash
git clone https://github.com/tolmachevmaxim/ai-news-digest.git
cd ai-news-digest
pip install -e .
claude
```

Then just say: **"Set up a news digest for me"**

Claude reads the project's `CLAUDE.md` and guides you through a conversational onboarding — asks about your topics, helps create a Telegram bot, learns your writing style from examples you paste, and runs a test digest. You can go off-script, ask questions, change your mind.

### Option B: CLI setup (no Claude Code)

```bash
git clone https://github.com/tolmachevmaxim/ai-news-digest.git
cd ai-news-digest
pip install -e .
python -m news_digest setup
```

The setup wizard walks you through:
1. **LLM backend** — Anthropic API key or Claude Code CLI
2. **Telegram bot** — step-by-step guide + automatic verification
3. **Topics & feeds** — pick from 10 preset packs or add custom RSS
4. **Your writing samples** — paste 3-5 examples, AI generates your style profile
5. **Preferences** — language, digest size, models

### Run

```bash
# Preview without sending
python -m news_digest run --dry-run

# Publish to Telegram
python -m news_digest run
```

### 4. Schedule (optional)

Run daily with cron:
```bash
# Every day at 8:00 AM
0 8 * * * cd /path/to/ai-news-digest && python -m news_digest run
```

Or macOS launchd:

## Configuration

All settings in `config.yaml` (generated by setup). Key fields:

| Field | Description | Default |
|-------|-------------|---------|
| `anthropic_api_key` | Anthropic API key | required |
| `telegram_bot_token` | Telegram bot token | required |
| `telegram_channel` | Channel ID (`@name` or `-100...`) | required |
| `feeds` | RSS/Atom feeds (name: url) | from topics |
| `keywords` | Relevance filter words | `[]` (keep all) |
| `language` | Digest language (`en`/`ru`) | `en` |
| `digest_size` | News items per digest | `7` |
| `selector_model` | Fast model for selection | `claude-haiku-4-5-20251001` |
| `writer_model` | Quality model for writing | `claude-sonnet-4-20250514` |
| `style_profile` | Auto-generated from your samples | from setup |
| `style_samples` | Your writing examples | from setup |

Environment variable overrides: `ANTHROPIC_API_KEY`, `TELEGRAM_BOT_TOKEN`.

### Securing your tokens

`config.yaml` stores tokens in plain text. This is fine for local use, but consider these options for better security:

**Environment variables** (recommended for servers/CI):
```bash
export ANTHROPIC_API_KEY="sk-ant-..."
export TELEGRAM_BOT_TOKEN="123456:ABC..."
# config.yaml values will be overridden
```

**macOS Keychain:**
```bash
# Store
security add-generic-password -a "$USER" -s "news-digest-anthropic" -w "sk-ant-..."
security add-generic-password -a "$USER" -s "news-digest-telegram" -w "123456:ABC..."

# Read in a wrapper script
export ANTHROPIC_API_KEY=$(security find-generic-password -s "news-digest-anthropic" -w)
export TELEGRAM_BOT_TOKEN=$(security find-generic-password -s "news-digest-telegram" -w)
python -m news_digest run
```

**Linux (secret-tool / pass):**
```bash
# GNOME Keyring
secret-tool store --label="news-digest" service anthropic-api-key <<< "sk-ant-..."
export ANTHROPIC_API_KEY=$(secret-tool lookup service anthropic-api-key)

# pass (GPG-encrypted)
pass insert news-digest/anthropic-api-key
export ANTHROPIC_API_KEY=$(pass news-digest/anthropic-api-key)
```

**Never commit** `config.yaml` to git — it's already in `.gitignore`.

## Cost

Per digest run:
- Selection (Haiku): ~$0.01
- Writing (Sonnet): ~$0.03-0.05
- **Total: ~$0.04-0.06 per digest, ~$1-2/month at daily frequency**

Using Opus for writing: ~$0.10-0.15 per run, ~$3-5/month.

## How the humanizer works

The writing prompt includes 12 condensed anti-AI rules from [blader/humanizer](https://github.com/blader/humanizer) (based on [Wikipedia's "Signs of AI writing"](https://en.wikipedia.org/wiki/Wikipedia:Signs_of_AI_writing)):

- No significance inflation ("pivotal", "groundbreaking", "testament")
- No -ing analyses ("highlighting...", "showcasing...")
- No promotional language ("vibrant", "stunning", "nestled")
- No AI vocabulary ("Additionally", "crucial", "delve", "foster")
- No sycophantic tone ("Great question!", "I hope this helps!")
- Add soul: vary rhythm, have opinions, acknowledge uncertainty

Combined with your personal style profile, the result reads like a human wrote it.
